HTML

  • The body element represents the visible content shown to the user.

CSS

  • While HTML strictly handles the structure and content of a page, CSS defines colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS, and External CSS style sheets.
  • Inline CSS is used to style a element directly within the HTML file, and will apply directly to a specific HTML tag.
  • Inline CSS styles are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
  • An example of Inline CSS would be "h3 style="color: red;">Prework Study Guide /h3"
  • It is possible to embed an entire stylesheet directly in the HTML file using a "style" element in the "head" element.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules will only apply to that page, and you can configure CSS
    classes and IDs, which are used to identify elements, to style multiple leements in the page code.
  • The most common way to style a webpage is by using External CSS style sheets, because it allows developers to keep all their CSS rules in a seperate file.
  • When using an external style sheet, we link the file using a "link" element place in the HTML's "head" element.
  • A link element will create the the connection to external files.
  • The rel attribute specifies the relationship between the current document and the linked document relationship.
  • The href attribute specifies the location (URL) of the external resource. It can also use a relative path. This means the location is derived from the current location, in this case, index.html.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class is defined using "class=xyz" and then recalled in CSS using the ."class name" to apply the properties to the element wearing the class. The I.D. for class will
    will identify where the class is being used. I.e. class="name" id="place to be used" and wrapped in your tag.
  • The border property will add a border to the element applied to and is sized using px and then a color defined.
  • The box-shadow property will add an outside shadow border to the border or element it is applied to. It is described by an x and y offset relative to the element applied to.
    These offsets are set in px and then a color defined.

GIT

  • All commands for GIT begin with "git" in CLI.
  • git status: check what branch we are currently working in.
  • git checkout -b "branch-name": create a new branch and switches to it.
  • git checkout "branch-name": will checkout a pre-existing branch to work out of.

Javascript

  • Using the console.log("name") in your javascript file is vital to debugging and testing your code.
  • .js files indicate that it is a Javascipt file.
  • The script element is used to link the scripts to the HTML index file. Take note that the script element needs a closing tag unlike the link element.
  • The script element will also follow underneath the footer element to allow the website to load, and then the scripts to take place.
  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • If statements will check whether a condition evaluates to true or false.
  • Javascript is read from top to bottom, and the order counts here, meaning we must be intentional about the we set up code in a JS file.
  • Control flow can be interrupted, however, we can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
